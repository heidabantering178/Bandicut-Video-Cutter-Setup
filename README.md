## 📝 Troubleshooting

If you encounter issues, look here for quick help.

* **File does not open:** Verify that the file format is supported. Ensure the file is not corrupted by trying to play it in your default media player first.
* **Audio and video out of sync:** This is rare in lossless mode. If this happens, ensure you are not changing the encode settings. Keep settings on default for the best result.
* **Crashing during save:** Make sure your hard drive is not full. The software needs enough room to write the new file before it removes the temporary data.
* **App remains slow:** Disable background tasks that consume CPU power. High-resolution files require more processing time, so allow the software to finish the task without interruption.
